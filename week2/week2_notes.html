WDD 330: Week 2 Notes

Programming basics that EVERYONE ought to know by now

Comments: single line comments are made by using the double slash '// '. Multi-line comments are made by using a combination of '/*' and '*/'.

Grammar: obviously use a ';' at the end of each line of code, separate each line of code onto its own line. A block is a series of statements that are inside of to curly brackets, '{ }'.

Primitive Data Types: String, Symbol, Number, Boolean, Undefined, and Null. Everything else is considered an object and follows the object rules.
