Chapter 8: CSS3 Transforms and Transitions

Transforms: The CSS3 transform property lets you translate, rotate, scale, and/or skew any element on the page.

Translation: Allows you to move an elements left, right, up, or down. These functions are similar to the behaviour of position when declaring top/left.

Transofrms and Older Browsers: To make transforms work in older browsers you can use webkits. '-webkit-transform' for android, '-ms-transform' for IE9 only. Also note that transforms don't work on inline elements. You can fix this by adding 'display: inline-block'.

Scaling: scaling function allows you to grow or shrink an element. If only one value is provided it will be used for both the x and the y values, growing or shrinking your element or pseudo-element while maintaining the original aspect ratio.

Rotation: Rotates an element around the point of origin by a spcified angle value. By default the point of origin is the elements center.

Skew: Specifies a skew along the x and y axes. Skew can be like rotate except that its more like looking at the object from a different perspective.

Transform Origin: Function can be used to change the origin of the transform.

Keyframes: allow you to animate objects using css. Use the keywords from and to.

Moving On: Use animations where necessary to enhance the user experience, everywhere else just skip it.

Chapter 12: Canvas, SVG, and Drag and Drop

Canvas: allows you to draw anything you can imagine on javascript.

Creating a canvas element: add a canvas element to the page, '<canvas> <canvas/>'. The canvas is then given through javascript. width and height must be added in the element like for a picture.

SVG: a picture made by vector graphics. You can zoom in on them as far as you want and they will keep their quality.

Whats XML: Stands for eXtensible Markup Language.

Much like the Canvas element, the SVG element needs to be declared with an svg tag.

SVG filters: Go to Inkscape, then choose color, moonarize.

Drag and Drop: Javascript has a library for this now that makes it super easy. Set an item to be draggable, give it an event listener for the dragstart event. Add event listeners for dragover and drop. Use the DataTransfer Object to specify an exchenge of information.
