Object Methods: this

Methods: Objects can act or have functions if you assign them to. For instance- "Object.funtion = function() {console.log("hi")}" is one way you could assign a function to an object. When we use objects to represent entities it is called object-orented-programming.

This: the this key word is used inside of object methods to refer to the object that is calling the function.

Arrow Functions: They do not have their own 'this' key word. In the function || let user = {firstName: "Ilya", sayHi() { let arrow = () => alert(this.firstName); arrow(); }}; || The 'this' key word refers to the outer function of 'sayHi'. This seems somewhat obvious to me as when you call an arrow function if you just use a parameter such as || (a) => function(a) || a would be the 'this' in this case.
